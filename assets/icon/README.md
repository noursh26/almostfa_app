# App Icon Setup

Place your app icon PNG files here:

1. `app_icon.png` - Main app icon (1024x1024 px recommended)
2. `app_icon_foreground.png` - Foreground for adaptive icon (1024x1024 px, with padding)

Then run:
```bash
flutter pub run flutter_launcher_icons
```

This will automatically generate all required icon sizes for Android.
