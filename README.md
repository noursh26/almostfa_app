## 🚀 كيفية البناء

### محلياً:
```bash
# تثبيت الحزم
flutter pub get

# بناء APK Debug
flutter build apk --debug

# بناء APK Release
flutter build apk --release --no-tree-shake-icons

# بناء AAB (لـ Google Play)
flutter build appbundle --release --no-tree-shake-icons
```

### عبر GitHub Actions:
1. ارفع المشروع على GitHub
2. اذهب إلى **Actions** → سيبدأ البناء تلقائياً
3. حمّل APK و AAB من **Artifacts**
4. للنشر: أنشئ Tag بصيغة `v1.0.0` وسيُنشأ Release تلقائي

### عبر Codemagic:
1. اربط المشروع بـ Codemagic
2. سيبني تلقائياً عند كل Push إلى `main`

## 🖼️ إضافة أيقونة التطبيق

### الخطوات:
1. ضع ملف `app_icon.png` (1024×1024 بكسل) في `assets/icon/`
2. ضع ملف `app_icon_foreground.png` (1024×1024 بكسل) في `assets/icon/`
3. شغّل الأمر:
```bash
flutter pub run flutter_launcher_icons
```

> 💡 **ملاحظة**: ملف `app_icon_foreground.png` يُستخدم للأيقونة التكيفية (Adaptive Icon) في Android 8+.
> يجب أن يحتوي على الشعار فقط مع هوامش كافية (padding ~25%).

## 🔑 توقيع التطبيق (Release Signing)

لتوقيع التطبيق للنشر على Google Play:

1. أنشئ Keystore:
```bash
keytool -genkey -v -keystore almostfa-release.jks -keyalg RSA -keysize 2048 -validity 10000 -alias almostfa
```

2. أنشئ ملف `android/key.properties`:
```properties
storePassword=YOUR_STORE_PASSWORD
keyPassword=YOUR_KEY_PASSWORD
keyAlias=almostfa
storeFile=../almostfa-release.jks
```

3. ابنِ النسخة:
```bash
flutter build apk --release
```
